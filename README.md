# AI-Chatbot
AI chatbot agent that demonstrates sophisticated reasoning, bias mitigation, and multi-perspective analysis when discussing politics.

How to use it:
1. Open AI_CHATBOT.ipynb.
2. Go onto OopenRouter and create an API key for the openai/gpt-4o-mini model and place it in OPENROUTER_API_KEY as indicated in the code.
3. Go onto Tavily and create an API key for the Tavily search functionality. Place it in Tavily_API_KEY as indicated in the code.
4. Run all the cells in order. The Gradio Integration cell must be actively running for the Gradio UI to work.
